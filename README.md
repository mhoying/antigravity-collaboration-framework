# Antigravity Global Environment Setup

This repository contains the foundational documentation, technical design, and system instructions for the Antigravity AI global environment setup.

## Files
- `requirements.md`: The Product Requirements Document (PRD) detailing functional needs, safety rules, and continuous learning constraints.
- `implementation_plan.md`: The Technical Design Document (TDD) that maps the PRD to the system architecture.
- `system_instructions.md`: The compiled system rules designed to be injected into the user's IDE Global Rules panel.
